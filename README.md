# 🍔 Simple Food

A graduation project built in **2025** for the **[From 0 to 1](https://from0to1.com.ua/)** course by **Vadim Prokopchuk**.

![Project collage with different screens](collage.jpg)

## 🚀 Live Demo

👉 [https://sfood-umber.vercel.app/](https://sfood-umber.vercel.app/)

## ✨ Features

- Clean and modern UI
- Fully responsive layout
- Tab-based navigation
- Interactive sliders

### Homepage

- Product category tabs
- Restaurant carousel
- Testimonials slider

### Catalog Page

- Custom checkbox filters
- Select inputs
- Price range slider

### Product Page

- Image gallery modal with slider
- Quantity selector with min/max limits and step controls
- Tabbed product details
- Star rating system in reviews section

## 🛠️ Tech Stack

- HTML
- SCSS
- JS (Vanilla)
- Gulp

## 📚 About

This project was created as the final assignment for the **From 0 to 1** course. During development, the project went through code reviews, helping improve code quality and overall development practices.
